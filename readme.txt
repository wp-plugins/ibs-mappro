Implementation of a Google Maps API v3 editor.

== Description ==

* A comprehensive Google Maps API v3 map editor for creating, viewing and maintaining maps. Supports kml and gpx map file formats.

* Customizable map marker icon palette and custom map marker icon libraries.

* Edit maps from plugin map folder, upload from desktop or load from the web.

* Create and print cue sheets using Google directions service.

* Add placemarks with links and images.

* Show elevation profiles

*Edit at map point level.

 

== Installation ==
1. Download ibs-mappro-0.1.zip and unzip.
2. Upload `ibs-mappro folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the ‘Plugins’ menu in WordPress
4 Admin | Settings menu | IBS Mappro and configure the plugin.
